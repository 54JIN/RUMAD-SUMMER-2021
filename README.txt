1.) When cloning or pulling from Github, make sure to type on the terminal (npm i) after going into the appropriate directory to install all the necessary npm modules.

2.) To Run Nodemon as a developer type (on the terminal) npm run dev. [This will run the index.js file contained within the src folder.]

3.) Create a folder named: config (with that exact name, no capitalization of any letter within the root directory - NOT inside the src folder) and then create a file named dev.env to include the following key value pairs with no space in between the equal signs: PORT, MONGODB_URL (Ex: PORT=3000 /*Press the enter button for next line*/ MONGODB_URL=/*the api key*/)
